# 앞으로의 공부방향

- 지금까지 공부한 내용 요약
  1. 재사용 가능한 컴포넌트를 만듭니다.
  2. props는 부모에게서 전달받는 값
  3. state는 자기 자신이 지니고 있는 데이터
  4. props나 state가 바뀌면 컴포넌트는 리렌더링 합니다.
  5. LifeCycle API를 통해서 컴포넌트 마운트, 업데이트, 언마운트 전후로 처리 할 로직을 설정하거나 리렌더링을 막아줄 수도 있다.
